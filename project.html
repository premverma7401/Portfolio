    ETimesheet
    (Under development)

    More than just a Timesheet Manager, ETimesheet help users to send their timesheet in few clicks from anywhere.
    No more Excel formatting and attachment issues. Users can even customize templates according to their preferences.

    Main Features:
  • Easy Login and Registration
  • View previously sent timesheets
  • Quick send, avoids retyping details
  • Save for Later, come back and fill it anytime
  • Notifications and reminders on your phone

    • Date: June 11, 2020
    • Technology Used: React Hooks, .NET Core Web API, MS Sql Server
    • Client: Open Source
    • Live View: premsager.com/etimesheet

    Payslip Generator

    Payslip generator is a simple tool for Payroll team to generate payslips for there employees based on the invoice or timesheet they receive. The tool can be used independent with only requirement to create Employees with in the application.

    Main Features:
  • Create Users in app
  • Weekly, Fortnightly or Monthly payslips
  • Auto Calculations on backend side
  • PDF download option
  • Web Preview option with 1-Click print

    • Date: March 11, 2020
    • Technology Used: ASP.NET Core MVC, Bootstrap 4
    • GitHub: View Code
    • Live View: premsager.com/payslip
